热门国产影片纯静态网站

生成结果：
- 影片数据：2000 条
- 独立详情页：2000 个，文件名为 movie-1.html 到 movie-2000.html
- 分类页：8 个，另含 categories.html、ranking.html、all-movies.html、search.html
- 所有 HTML 页面已插入百度统计脚本。
- 页面封面按要求引用根目录 1.jpg 到 150.jpg：第 N 部使用 ((N - 1) % 150) + 1.jpg。

使用方法：
1. 将 1.jpg 到 150.jpg 放到网站根目录。
2. 将整个文件夹上传到任意静态空间即可访问。
3. 首页入口为 index.html。
